```math
\begin{cases} \log_{e}(x) = z \cdot 1 \leftarrow e^{z^{1}} = e^{z \cdot 1} = x\\
\log_{e}(x^{y}) = z \cdot y \leftarrow e^{z^{y}} = e^{z \cdot y} = x^{y} \end{cases}
```
